Using Angular

Angular has three main components related to your app, after initialization these are all found within the src/app folder of your project. If the server is currently running your created app, any modifications to these files will be rendered instantly after saving the document.

Modify template

To modify the display of your page, you will need to access the "app.component.html". From here you can insert html tags to be rendered by the browser.

Display Variables

You can display variables defined inside "app.component.ts" directly inside the "app.component.html", by enclosing them inside of double curly braces.
